## 🎯 How to Use

### **Quick Start (5 minutes)**
```bash
npm run setup
cp .env.example .env
npm run server:dev    # Terminal 1
npm run client:dev    # Terminal 2
# Open http://localhost:5173
```

### **Admin Access**
- URL: http://localhost:5173/admin/login
- Email: admin@glassshop.com
- Password: Admin@123

### **Add Your Data**
1. Create categories (Tempered Glass, Chargers, etc.)
2. Add products with prices and stock
3. Test with customer orders
4. Adjust bulk pricing if needed
