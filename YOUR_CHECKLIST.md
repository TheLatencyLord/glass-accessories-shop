## 📋 अभी आपको करना है

### **Week 1: Local Testing**

**Step 1: Install करो (30 min)**
- [ ] Node.js download और install करो (https://nodejs.org)
- [ ] MongoDB install करो (Mac: brew install, Windows: download, Linux: apt-get)
- [ ] `.env` file बनाओ और values fill करो

**Step 2: Run करो (10 min)**
- [ ] Terminal 1: `npm run server:dev` (Backend start)
- [ ] Terminal 2: `npm run client:dev` (Frontend start)
- [ ] Browser में खोलो: http://localhost:5173

**Step 3: Test करो (2-3 hours)**
- [ ] Homepage देखो
- [ ] Products browse करो
- [ ] Admin login करो
- [ ] Categories add करो
- [ ] Products add करो
- [ ] Bulk orders test करो (10+ items डालो)
- [ ] Checkout complete करो
- [ ] WhatsApp links test करो

### **Week 3: Deployment Preparation**

**Step 6: Infrastructure खरीदो**
- [ ] **Domain खरीदो** (₹200-500/year)
  - [ ] GoDaddy: https://www.godaddy.com
  - [ ] Namecheap: https://www.namecheap.com
  - [ ] Google Domains: https://domains.google
  
  Suggested name: glassaccessoriesindia.com (या आपका business name)

- [ ] **Hosting खरीदो** (₹1000-5000/month)
  - [ ] Option A: **Vercel** (Recommended - सबसे easy)
    - https://vercel.com
    - Pay as you go (~₹0-2000/month)
  
  - [ ] Option B: **Railway.app** (New aur good)
    - https://railway.app
    - Pay as you go (~₹500-1000/month)
  
  - [ ] Option C: **AWS** (Powerful but complex)
    - EC2 + RDS (~₹3000-5000/month)
  
  - [ ] Option D: **Heroku** (Simple but paid)
    - https://heroku.heroku.com
    - Hobby tier (~₹500-1000/month)

- [ ] **Database (Cloud)** (₹0 recommended)
  - [ ] MongoDB Atlas (Free tier - 512MB)
    - https://www.mongodb.com/cloud/atlas
  - [ ] यह एकदम पर्याप्त है starting के लिए
